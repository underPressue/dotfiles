# Yazi Configuration

## Tokyo Night Theme

This configuration uses the Tokyo Night theme for yazi.

### Initial Setup

The Tokyo Night flavor needs to be cloned separately:

```bash
mkdir -p ~/.config/yazi/flavors
cd ~/.config/yazi/flavors
git clone https://github.com/BennyOe/tokyo-night.yazi.git tokyo-night.yazi
```

Or use the yazi package manager:

```bash
ya pkg add BennyOe/tokyo-night
```

### Theme Location

- Flavors directory: `~/.config/yazi/flavors/tokyo-night.yazi/`
- Active theme: `tokyo-night`

### Files

- `theme.toml` - Configures the Tokyo Night flavor
